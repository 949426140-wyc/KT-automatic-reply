# 项目约定

- 项目名称：产品自动回复
- 项目根目录：`D:\AI\产品自动回复`
- 本项目后续产生的配置、设置、项目记忆、中间产物和最终文件，统一保存在该目录及其子目录中。
- 除非用户另有明确指定，不在其他位置存放本项目文件。

## 自动回复维护边界

- 本地 JS 负责问题类型过滤、关键词质量过滤、平台连接、消息收发、去重和日志。
- “建议 / 通常 / 一般”等词是否允许或拦截，直接在本地 `钉钉自动回复\auto-reply.js` 中维护。
- Dify 知识库 Markdown 只维护产品相关知识，包括产品事实、规格尺寸、材质、安装、选型、兼容关系和具体产品标准答案。
- 不在 Dify 产品知识库中维护通用过滤词清单，也不把订单、物流、售后等通用问题类型过滤规则混入产品知识卡。
- 修改本地 JS 后重建机器人并验证；修改产品知识后同步 `_Dify上传合集`，由用户重新上传 Dify。
