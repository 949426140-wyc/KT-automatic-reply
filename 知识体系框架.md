## 2. 产品知识体系（四层结构）

### L1 底层物料与标准

#### 核心材质

| 材质 | 应用范围 | 核心特性 |
|------|----------|----------|
| 抗倍特板（10mm） | 四代魔法抽侧板 | 高硬度、防潮防水、耐化学腐蚀、E0级环保、阻燃、抗菌易清洁 |
| 六系航空铝（6061/6062） | 尚酷抽侧板、横杆 | C919机身同系材料，壁厚≥1.2mm（关键承重区1.4mm+），肤感喷涂 |
| 海蒂诗 Quadro YOU 轨道 | 所有抽屉 | 纯进口（非珠海产），8万次推拉测试，标配250-450mm |
| 铸铝连接件 | 抽屉四角 | 结构稳定性+200%（vs竞品塑料件），成本3-5倍 |

**竞品对标**：酷太（Quadro YOU进口+铸铝）> 巴斯蒂姆（Actro定制+塑料）> 诺米（赫马国产+塑料）> 悍高（安格斯国产+塑料）

#### 基础尺寸常数

| 常数项 | 魔法抽 | 尚酷抽 |
|--------|--------|--------|
| 侧板厚度 | 10mm | 11.5mm |
| 轨道间隙（单侧） | 11mm | 9.5mm |
| 底板厚度 | 9mm | 9mm |
| 底板间隙 | 15mm | 15mm |
| 铰链垫块（开门款） | 25mm | 25mm |
| 柜体侧板标准 | 18mm | 18mm |
| 抽中抽高度 | 55mm | - |

#### 核心公式

- **抽屉外宽（连门款）** = 柜内净宽 - 轨道间隙×2
- **抽屉外宽（双开门款）** = 柜内净宽 - 轨道间隙×2 - 铰链垫块×2
- **内部可用宽** = 抽屉外宽 - 侧板厚度×2（魔法抽-20mm / 尚酷抽-23mm）
- **内部净高** = 抽屉侧板高 - 底板(9mm) - 底板间隙(15mm) = 侧板高 - 24mm
- **抽中抽下层净高** = 侧板高 - 24mm - 55mm
- **轨道深度** = 柜内净深 - 50mm → 选最近标准轨道
- **抽屉深度** = 轨道深度 + 30mm（标准品480mm = 轨道450mm）
- **内部模块长度** = 抽屉深度 - 33mm

#### 标准轨道系列

| 轨道深度 | 系列 | 说明 |
|----------|------|------|
| 250-450mm | Quadro YOU | 标配，450mm=抽屉480mm深 |
| 500-550mm | Quadro YOU | 加长 |
| 550-600mm | Quadro V6 | 中载 |
| 600-650mm | Actro | 重载，+100元起 |

### L2 核心功能组件

#### 抽屉五大结构类型

| 结构 | 核心机制 | 适配物品 | 限制 |
|------|----------|----------|------|
| **横向分隔抽** | 双横杆+小分隔片四面可装，前后左右均可调节 | 调料瓶袋、食品零食（规格多变的物品） | 单格空间有限 |
| **纵向分隔抽** | 单横杆（近门口）+前排小格+后排长通道 | 锅具竖插、茶叶罐、洗护品（规格统一的物品） | - |
| **抽中抽** | 内侧加轨道叠一层薄抽(55mm)，一层变两层 | 矮物品上方空间利用 | 最小外轨350mm，内轨200mm起 |
| **半抽** | 中间加侧板一分为二，一半双层+一半单层 | 碗碟高矮兼顾（南方最畅销） | 最小宽500mm，最小轨400mm |
| **子母抽** | 独立两层，磁吸联动，门板可倾斜打开 | 日式紧凑空间（仅厨房、仅270H） | 空间拆得碎，不能插放 |

|------|----------|----------|

| 130H | 小瓶调料、饮料、茶叶罐、工具抽中抽 | 无抽中抽 |

| 230H | 碗碟竖插、锅具竖插、调料分隔（主力高度） | 有抽中抽（下层净高160mm） |
| 270H | 倾松抽（子母抽） | 仅子母抽 |
| 550H | 调料整抽（一体式，三层四区） | 无（竖抽） |

### L4 设计理念与用户画像

#### 核心设计哲学

- **四步原则**：物有所藏 → 归有其位 → 收取便捷 → 健康安全
- **核心指标**：3秒拿到、平铺收纳、分区可视化、空间利用率+30%
- **两全三高四真**：全屋+全人群、高颜值+高品质+高体验、真场景+真需求+真痛点+真方案

## 3. 客服运营体系

### A. 订单履约流（正向）

#### 下单前

| 校验项 | 规则 | 决策依据 |
|--------|------|----------|
| 尺寸定制 | 宽度200-1200mm / 高度70-280mm（魔法抽） | 柜内净宽-轨道间隙-垫块=抽屉外宽 |
| 深度定制 | 非480标品需加收深度定制费 | 轨道深度=柜内净深-50mm |
| 颜色确认 | 魔法抽奶咖色 / 尚酷抽雪花灰 / 墙面产品统一同色系 | 所有产品颜色为同一配色体系 |
| 价格校验 | 以圆方系统报价为准（非有赞价格） | 8折为底线，低于8折需审批 |
| 连门/开门 | 连门款：前挡板铝合金横条 / 开门款：完整前挡板+垫块 | 开门款需避让铰链 |
| 合规签署 | 小B端需签署《合作经营承诺书》+《不乱价承诺》 | 签署盖章回传后才审核订单 |

#### 发货中

| 规则项 | 标准 |
|--------|------|
| 标品时效 | 48小时发货（标签为"48小时发货"的产品） |
| 定制时效 | 7-10个工作日（含定制件产品） |
| 物流方式 | 德邦/顺丰，默认送货上楼（有电梯） |
| 偏远地区 | 不发货/特殊处理 |
| 免费寄存 | 7天（完成后可通知发货） |
| 延迟发货 | 支持的，备注发货日期或群内通知即可 |
| 拆分发货 | 部分缺货可先发有货品，缺货品到货补发 |
| 物流查询 | 钉钉群→输入框上方"物流查询"→输入订单号 |

#### 物流时效参考

| 区域 | 时效 |
|------|------|
| 江浙沪皖 | 1-2天 |
| 华北、华南 | 2-3天 |

### B. 售后服务流（逆向）

#### 第一决策：问题定性

| 责任方 | 典型场景 | 处理原则 |
|--------|----------|----------|
| 总部责任 | 质量问题、漏发、错发、物流破损 | 免费补发/换新，总部承担运费 |
| 服务商/客户责任 | 安装损坏、需求变更、使用不当（液体浸泡等） | 付费补发/购买，客户承担运费 |
| 待判定 | 导轨生锈/腐蚀、底板鼓泡 | 看图片判断：液体腐蚀→人为；自然生锈→质量问题 |

| 配件类型 | 包含项目 | 处理方式 |
|----------|----------|----------|
| 小件 | 螺丝、分隔件、连接件 | 无需寄回旧件，直接免费补发（总部责任） |
| 大件 | 骨架、轨道、挂盒 | 需先发后取/先下单后退款，涉及财务风控 |

## 4. 高频场景决策逻辑

### 场景一："X产品能做吗？"

```
物品识别 → 需求转译 → 载体匹配 → 尺寸校验
   │            │           │           │
   │            │           │           └→ 宽200-1200? 高70-280? 轨250-650?
   │            │           └→ 地柜→抽屉 / 吊柜→升降机 / 高柜→中枢阁
   │            └→ 碗碟→碗碟类 / 锅具→锅具类 / 调料→调料类
   └→ 客户想放什么? 插放还是堆放?
```

### 场景二："售后问题怎么处理？"

```
责任判定 → 配件类型 → 流程选择
   │           │           │
   │           │           ├→ 总部+小件 → 免费直发
   │           │           ├→ 总部+大件 → 先发后取/先下单后退款
   │           │           └→ 客户责任 → 付费链接
   │           └→ 螺丝/分隔片=小件 | 骨架/轨道/挂盒=大件
   └→ 看图片：物流破损?液体腐蚀?安装不当?产品瑕疵?
```

### 场景三："发货到哪了？"

```
查状态 → 定位节点 → 原因解释
   │           │           │
   │           │           └→ 定制未到7-10天? 节假日顺延? 拆分漏发?
   │           └→ 待审核/生产中/待发货/已发货/已签收
   └→ 钉钉物流查询 → 输入订单号
```

### 场景四："价格/合同问题"

```
确认诉求 → 查系统 → 给解释
   │           │           │
   │           │           └→ 圆方报价为准 / 折扣审核 / 合同签署状态
   │           └→ 圆方查价 / 订单查折扣 / 查签署状态
   └→ 问报价? 问折扣? 问合同?
```

## 5. 安装知识与异常处理

### 安装方式选择

#### 连门款 vs 开门款

| 属性 | 连门款 | 开门款 |
|------|--------|--------|
| 结构 | 门板直接装在抽屉前挡板上 | 保留原柜门（铰链门），抽屉独立 |
| 前挡板 | 两条六系铝合金横条（预留螺丝孔） | 完整前挡板（美观） |
| 使用方式 | 拉抽屉=开门 | 先开门再拉抽屉 |
| 轨道安装 | 轨道直接装侧板 | 需先装25mm垫块避让铰链 |
| 轨道间隙 | 11mm/侧 | 11mm + 垫块厚度/侧 |
| 安装难度 | 简单 | 较复杂 |

#### 垫块规格

| 厚度 | 适用场景 |
|------|----------|
| 2mm | 微调 |
| 10mm | 窄缝隙 |
| 15mm | 中等 |
| 18mm | 标准（与柜侧板同厚） |
| 25mm | 标配开门款，避让铰链 |

**特殊安装配件**：木条（任意厚度补充）、L型底座垫块（柜体远超抽屉宽度时）、造型模板紧固件（造型门板螺丝偏转）、一字连接片（双门板连接）、甲壳虫反弹器

### 特殊柜体安装

| 柜体类型 | 处理方式 |
|----------|----------|
| 麻石柜体 | 较软，常规处理 |
| 大理石柜体 | 常规处理 |
| 花岗岩柜体 | 极硬，需专用钻头 |
| 燃气灶下抽屉 | 部分地区强制门板通风孔 |
| 上翻门吊柜 | 测量最低点（深度+高度二维坐标） |
| 分水器柜 | 分段测量深度，判断上方是否有可用空间 |

### 常见安装异常

#### 抽屉类

| 异常 | 可能原因 | 排查方向 |
|------|----------|----------|
| 推拉异响 | 轨道安装不平、侧板挤压、异物卡入 | 检查轨道水平度、侧板间隙是否均匀 |
| 抽屉关不严 | 轨道限位失效、物品顶住 | 检查轨道末端限位器、清理异物 |
| 抽屉自行滑出 | 地面不平、轨道前倾 | 调整脚垫使柜体水平 |
| 抽中抽卡顿 | 内轨装反、净高不足 | 确认内轨方向，核实下层净高 |
| 开门款抽屉与门板干涉 | 垫块厚度不够、铰链型号不匹配 | 加厚垫块或更换铰链 |

#### 升降机类

| 异常 | 可能原因 | 排查方向 |
|------|----------|----------|
| 云狐升降无力/弹簧感 | 2.0机构力度需调节 | 运行中旋转调节钮（2.0支持实时调） |
| 尚酷升降机无法调节 | 1.0机构未复位 | 必须先复位到底再调节，否则损坏弹簧 |
| 云梯无响应 | 电源线松动、限位丢失 | 检查电源，重新设置上下限位 |
| 云梯感应失灵 | 背板间距不足 | 背板与两侧各留1-2cm |
| 云狐误触发防夹 | 底板传感器被异物遮挡 | 清理底板传感器表面 |

#### 置物架类

| 异常 | 可能原因 | 排查方向 |
|------|----------|----------|
| 中枢阁极窄柜阻力大 | 阻尼器力矩过大 | 极端情况可拆卸底部三轨阻尼器 |
| 翼枢阁折叠不顺畅 | 铰链螺丝过紧 | 适当放松铰链螺丝 |
| 挂门宝门板倾斜 | 单点承重、长期受力 | 确认侧板承重安装正确、挂盒分布均匀 |

### 安装异常判定流程

```
客户报异常 → 要照片/视频 → 判断类型
                              │
                    ┌─────────┼─────────┐
                    ↓         ↓         ↓
                 安装问题   产品问题   使用问题
                    │         │         │
             指导门店重装  售后补发   解释后付费换
             或联系师傅    免费换新   告知使用方法
```

**判断依据**：
- 安装问题：抽屉不平、轨道未对齐、垫块缺失
- 产品问题：材质缺陷、焊接开裂、轨道出厂故障
- 使用问题：液体浸泡痕迹、碰撞凹陷、自行改装

### 组合方案设计原则

5. **深度匹配柜体**：标准480mm，定制根据柜深计算

## 7. 客诉处理体系

### 处理原则

1. **先听完整再判断**：不打断客户，获取全部信息（图片/视频/时间线）
2. **先定性责任再给方案**：总部原因 vs 服务商/客户原因，路径不同
3. **不承诺做不到的事**：不确定的事情说"我帮你确认"而非"没问题"
4. **一次给完整方案**：告知处理步骤+预计时效+需要客户配合的事项

### 典型客诉场景处理

#### 产品故障投诉

```
客户：产品质量有问题，要求免费更换

判断流程：
1. 索要产品图片/视频，拍摄角度：全景+细节+标签
2. 判断损坏性质：
   - 材质开裂、出厂瑕疵 → 属于质保范围，安排更换
   - 液体浸泡痕迹（底板鼓泡、导轨生锈）→ 属于使用不当
3. 回复话术：
   质保范围："看到图片了，这个属于质量问题，我帮您安排免费更换，预计X天发出"
   使用不当："从图片看是因为XX导致的XX，不属于免费质保范围。
             不过可以单独购买配件替换，帮您查下费用"
```

#### 发货延迟投诉

```
客户：怎么还不到！客户在催！

处理流程：
1. 查订单状态（待审核/生产中/待发货/已发货）
2. 定位原因
3. 回复话术：
   生产中："这个订单含定制件，定制时效是7-10个工作日，目前是第X天，预计X号左右完成"
   已发货："已发出，这是快递单号，目前显示XX（物流状态），预计X号前到"
   未审核："这个订单还未审核，需要先XX（签合同/补齐信息），完成后立即处理"
```

#### 价格争议

```
门店：怎么系统算出来比我手算的贵？

处理流程：
1. 确认报价以圆方系统为准
2. 逐项核对产品组件和单价
3. 如有差异，以系统为准并向门店说明
4. 回复话术："报价以圆方系统为准。我帮您逐项核对，目前系统报价XX，
          包含了XX费用（定制费/配件费），您手算的可能漏了XX项"
```

#### 物流损坏

```
客户：收到货发现产品变形/破损

处理流程：
1. 索要：快递面单照片+破损产品全景+细节+包装照片
2. 判断：物流责任 → 总部补发，同时与快递公司理赔
3. 回复话术："看到了，这个很明显是物流运输中造成的。
          我马上安排补发，预计X天发出，破损件不用退回。
          同时我们会与快递公司对接理赔，不需要您操作"
```
