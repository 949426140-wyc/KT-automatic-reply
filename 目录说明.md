# 目录与运行数据说明

本仓库仅保存钉钉自动回复服务的源代码、测试、部署配置和文档。

运行时依赖以下与仓库同级的数据目录，它们不会提交到 GitHub：

- `../产品知识库`：Obsidian 产品事实、结构化产品卡、客服规则和 LLM Wiki。
- `../图片库`：产品原图与图片索引。

其中客服规则与待确认记录位于 `../产品知识库/03_客服自动回复资料`；机器人可通过 `BOT_KB_DIR` 覆盖该位置。产品事实根目录可通过 `PRODUCT_KB_ROOT` 覆盖。

本地机密和运行状态（`.env`、`data/`、`runtime/`、日志、审计结果）均被 `.gitignore` 排除。首次部署请复制 `.env.example` 为 `.env`，再按 `README.md` 与 `服务器部署说明.md` 配置。
